# GripRush Privacy Policy

This repository hosts the public, canonical GripRush privacy policy via
GitHub Pages, linked from the Google Play Store listing.

Live page: https://baumi-x.github.io/griprush-privacy/

The source draft (with editorial history) is maintained in the main
GripRush game repository at `Griprush/PRIVACY_POLICY_DRAFT.md`. When that
draft changes, update `index.html` here to match and keep the "Last
updated" date in sync.
