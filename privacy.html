Kurzfassung

GripRush erhebt keine personenbezogenen Daten zur Identifizierung von Nutzern und überträgt keine Daten an den Entwickler oder Dritte. Welche Daten werden gespeichert?

GripRush speichert folgende Spieldaten lokal auf deinem Gerät (mittels localStorage, also im privaten App-Speicher deines Geräts/Browsers):

  • Coin-Guthaben
  • Gekaufte Shop-Upgrades (z. B. Speed, Grip, Coin-Multiplikator)
  • Highscores und Lauf-Historie (Punktzahl, Distanz, Combo, Zone, gewählter Charakter, Zeitstempel des Laufs)
  • Einstellungen (z. B. Sound an/aus)
  • Tutorial-Fortschritt (ob das Tutorial bereits gesehen wurde)

Diese Daten enthalten keine personenbezogenen Informationen (keinen Namen, keine E-Mail-Adresse, keine Kontaktdaten, keinen Standort, kein Foto-/Video-/ Mikrofon-Zugriff) und werden ausschließlich auf deinem eigenen Gerät verarbeitet.

Werden Daten an Server übertragen?

Nein. GripRush ist ein vollständig offline lauffähiges Spiel. Es findet keine Übertragung von Spieldaten an uns oder Dritte statt. Es gibt keine Server-Anfragen für Gameplay, Speicherstände oder Nutzerprofile.

Analytics, Werbung, Tracking

GripRush verwendet keine Analytics-Dienste (z. B. Google Analytics, Firebase Analytics), keine Werbenetzwerke und keine Tracking-/Fingerprint- Mechanismen. Es findet kein Abgleich mit Werbe-IDs statt.

Aktuell verwendete Dienste

Zum Stand dieser Datenschutzerklärung nutzt GripRush folgende Dienste bzw. Funktionen:

  • Analytics: keine
  • Firebase: keine
  • Werbung / AdMob: keine
  • Online-Funktionen: keine
  • Cloud-Speicherung: keine

Sollte sich dies in Zukunft ändern (z. B. durch Einführung eines Coin-Shops mit Online-Synchronisierung), wird diese Liste vor Veröffentlichung der jeweiligen Version aktualisiert.

Berechtigungen

GripRush benötigt keine besonderen Geräteberechtigungen (kein Zugriff auf Kamera, Mikrofon, Kontakte, Standort, Dateisystem außerhalb des eigenen App-Speichers).

Datenlöschung

Da alle Daten ausschließlich lokal gespeichert werden, kannst du sie jederzeit selbst löschen, indem du die App-Daten von GripRush über die Android-Systemeinstellungen löschst oder die App deinstallierst. Es gibt keine serverseitig gespeicherten Daten, die separat gelöscht werden müssten.

Kinder

GripRush richtet sich nicht gezielt an Kinder, fragt kein Alter ab und sammelt generell keine Daten. Da GripRush keine personenbezogenen Daten erhebt, keine Kommunikation mit anderen Nutzern ermöglicht und keine Werbe-/Tracking-Mechanismen enthält, bestehen aus datenschutzrechtlicher Sicht keine besonderen Risiken für jüngere Spieler.

Änderungen dieser Datenschutzerklärung

Sollte sich der tatsächliche Funktionsumfang von GripRush ändern (z. B. durch spätere Einführung von Online-Funktionen, Analytics oder Werbung), wird diese Datenschutzerklärung entsprechend aktualisiert, bevor eine solche Version veröffentlicht wird.

What data is stored?

GripRush stores the following gameplay data locally on your device (via localStorage, i.e. your device's/browser's private app storage):

  • Coin balance
  • Purchased shop upgrades (e.g. speed, grip, coin multiplier)
  • High scores and run history (score, distance, combo, zone, selected character, run timestamp)
  • Settings (e.g. sound on/off)
  • Tutorial progress (whether the tutorial has already been seen)

This data contains no personally identifiable information (no name, email address, contact details, location, or camera/microphone/photo access) and is processed exclusively on your own device.

Is data transmitted to any server?

No. GripRush is a fully offline-capable game. No gameplay data is transmitted to us or any third party. There are no server requests for gameplay, save data, or user profiles.

Analytics, advertising, tracking

GripRush does not use any analytics services (e.g. Google Analytics, Firebase Analytics), advertising networks, or tracking/fingerprinting mechanisms. No advertising ID matching takes place.

Currently used services

As of this privacy policy, GripRush uses the following services or features:

  • Analytics: none
  • Firebase: none
  • Advertising / AdMob: none
  • Online features: none
  • Cloud storage: none

Should this change in the future (e.g. by introducing a coin shop with online synchronization), this list will be updated before the respective version is released.

Permissions

GripRush does not require any special device permissions (no access to camera, microphone, contacts, location, or file system outside its own app storage).

Data deletion

Since all data is stored locally only, you can delete it at any time yourself by clearing GripRush's app data via Android system settings or by uninstalling the app. There is no server-side stored data that would need to be deleted separately.

Children

GripRush is not specifically directed at children, does not ask for age, and does not collect any data in general. Since GripRush does not collect personal data, does not enable communication with other users, and contains no tracking/advertising mechanisms, there are no particular privacy risks for younger players from a data-protection standpoint.

Changes to this privacy policy

If GripRush's actual feature set changes in the future (e.g. by introducing online features, analytics, or advertising), this privacy policy will be updated accordingly before such a version is released.
